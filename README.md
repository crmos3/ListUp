# Script

ToStringManager.csはpython_script/variable.pyを利用して書き出しています。  
variable.txtに書き出したいクラスのソースコードをコピペした上でvariable.pyを実行してください。標準出力にコードが出力されます。
正規表現が正確ではないため、正常な出力が得られない場合があります。  
最後のSeparatorは余計です。手作業で削除してください。
